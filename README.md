# ds-util

My util scripts for data science projects and Kaggle.

## Setup

```bash
poetry install
```

## Test

```bash
poetry run pytest -v
```

```bash
poetry run pytest -v --cov=ds_util
```

## How to install

```bash
pip install git+https://github.com/kiccho1101/ds-util.git

# or 

poetry add git+https://github.com/kiccho1101/ds-util.git
```
